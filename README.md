## Library

Install library with platformio

```bash
 //Board Arduino 
#include <Arduino.h>

//Library for motor
#include <AFMotor.h>
#include <Servo.h>

//Library for radio
#include <RF24.h>
#include <nRF24L01.h>
#include <SPI.h>

//library for display
#include <Adafruit_SSD1306.h>
#include <Wire.h>
#include <Adafruit_GFX.h>

```
